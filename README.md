## 📌 About the Project

This project is an **AI-powered cricket analytics tool** that predicts:

- 🟢 Safe score  
- 🔵 Expected score  
- 🔴 Aggressive score  

It also includes:

- 🌧️ Rain-shortened match adjustment  
- 📊 AI vs DLS-style comparison  
- 📈 Score distribution visualization  
- 🧠 Match insights (momentum, pressure, etc.)

- 🎯 ML-based score prediction (T20 & ODI)

- 📊 Interactive visual charts

- 🧩 Clean, professional UI (Streamlit)
- 🏏 Team selection (Local / IPL / International)

---

## 🧠 How It Works

The model takes inputs like:

- Current score
- Overs completed
- Wickets lost
- Recent performance (last 6 / 12 balls)

Then predicts final score ranges using trained ML models.

Rain adjustment is simulated by:
- Reducing total overs
- Recalculating scoring potential
- Comparing with DLS-style baseline
